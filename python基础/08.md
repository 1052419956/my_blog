## 前端三兄弟
----
* html 负责内容
    - 各种标签
    - 显示什么内容由html控制
* css 负责样式
    - 设置属性
    - 内容显示成什么样子，由css控制
* javascript 负责动态效果，数据处理（js）
    - 交互由js决定
    - js算是真正意义上的编程语言
